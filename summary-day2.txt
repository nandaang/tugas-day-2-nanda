Summary day 2:

Ini menjawab pertanyaan nomor 1 dan 3 sekaligus ya mas. :D

(init) initialized git
(touch) membuat folder baru
(status) untuk cek status git
(remote add origin) sinkronisasi link git dengan komputer
(remote -v) melihat link yang disinkronisasi
(pull origin master) tarik data dari ekspedisi
(add) menyiapkan yang akan dikirim ke ekspedisi
(commit -m "push") ekspedisi masukkan barang ke truk
(push origin master) barang dikirim truk ke penerima
